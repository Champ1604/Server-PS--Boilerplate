# Duskflare Server

This is a Clone of PS-Boilerplate.

## Quick Start

```bash
$ git clone https://github.com/Tigerball890/Server-PS--Boilerplate.git
$ cd Server-PS--Boilerplate && npm install
$ npm install cheerio
$ npm install request
$ node app.js
```
